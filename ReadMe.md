⭐ Quizify – Features
1. Quiz Categories

Quizzes are organized into multiple categories (e.g., General Knowledge, Science, Math).

Users can easily browse and select their preferred category.

2. Difficulty Levels

Each quiz can be taken at different difficulty levels:

Easy

Medium

Hard

Helps users choose quizzes based on their skill level.

3. Quiz Rules

Clear instructions displayed before starting a quiz.

Rules may include scoring format, time limits, negative marking, and navigation guidelines.

4. Timer per Quiz

A countdown timer is applied to the entire quiz.

Automatically submits the quiz when time expires.

Adds challenge and encourages quick thinking.

5. Score Calculation

Automatic score calculation based on correct and incorrect answers.

Displays:

Total score

Number of correct answers

Number of wrong answers

Accuracy percentage (optional)

6. Negative Marking

Supports deduction of marks for incorrect answers.

Penalty percentage/points can be configured based on the quiz.

7. Restart Quiz

Users can retake the quiz after completion.

Useful for practice and improvement.

8. Random Questions

Questions are randomized each time a quiz starts.

Helps avoid memorization and improves fairness.

9. Save Score History

User scores are saved for future reference.

Helps track progress over time.

Shows date, category, difficulty, and score details.

10. Next and Previous Navigation

Allows users to move freely between quiz questions.

Review and change answers before final submission.
